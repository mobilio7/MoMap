# 모빌리오 지도, 모맵입니다.

모맵은 모빌리오 로봇들의 움직임 관찰을 위해 만들어진 어플리케이션입니다. 

실시간 GPS 추적 기능을 활용하여 로봇이 아닌 핸드폰, 태블릿 등의 실시간 위치 추적도 가능합니다. 

이동 모드 버튼을 클릭하면 로봇 혹은 사용자의 실시간 위치를 추적하여 이동 경로를 그려냅니다.
